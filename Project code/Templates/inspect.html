📊 Model Information

🎯 Model Details
Model Type: Random Forest Classifier
Task: Fetal Health Classification
Classes: 3 (Normal, Suspect, Pathological)
Input Features: 21 Cardiotocogram Features
📈 Features Used
  • Baseline & Movement: Baseline value, Accelerations, Fetal movement, Uterine contractions
  • Decelerations: Light, Severe, and Prolongued decelerations
  • Variability: Abnormal short-term, Mean short-term, Abnormal long-term percentage, Mean long-term
  • Histogram: Width, Min, Max, Peaks, Zeroes, Mode, Mean, Median, Variance, Tendency
🏥 Prediction Classes
Class 1: ✅ NORMAL - Healthy fetal condition
Class 2: ⚠️ SUSPECT - Requires further evaluation
Class 3: ❌ PATHOLOGICAL - Serious abnormalities detected
💡 How It Works
  • Enter cardiotocogram (CTG) measurements from fetal monitoring
  • The model analyzes the input features using machine learning
  • Outputs a classification with confidence score
  • Results should be used as a decision support tool alongside clinical judgment
